## 3. 耳机端原理图详情

### 3.1 主控模块（Page 2）

```
                    Seeed XIAO ESP32C3
                    ┌──────────────────┐
         ICM-42688  │                  │
         SDA ──────►│ D0 (GPIO2)       │
         SCL ──────►│ D1 (GPIO3)       │
                    │                  │
         DFPlayer   │                  │
         TX  ──────►│ D2 (GPIO4)  UART │──► DFPlayer RX
         RX ◄──────│ D3 (GPIO5)  UART │◄── DFPlayer TX
                    │                  │
         SW1 ──────►│ D4 (GPIO6)       │  模式按键
         SW2 ──────►│ D5 (GPIO7)       │  VOL+
         SW3 ──────►│ D6 (GPIO10)      │  VOL-
                    │                  │
         TP4056     │                  │
         STDBY ────►│ D7 (GPIO20)      │  充电状态
                    │                  │
         LED ◄─────│ D8 (GPIO21)       │  状态指示
                    │                  │
         电池分压   │                  │
         ─────────►│ A0 (GPIO1)  ADC  │  电量检测
                    │                  │
         3V3 ──────│ 3V3              │
         GND ──────│ GND              │
         5V  ──────│ 5V               │
                    └──────────────────┘
```

**关键网络标号**:
- `I2C_SDA`, `I2C_SCL` — I2C 总线
- `UART_RX`, `UART_TX` — DFPlayer 通信
- `BTN_MODE`, `BTN_VOL_UP`, `BTN_VOL_DN` — 按键输入
- `CHG_STATUS` — 充电状态
- `LED_STATUS` — LED 驱动
- `BAT_SENSE` — 电池电压检测

## 6. PCB 布局注意事项

1. **音频走线**：DFPlayer 到 PAM8403 的音频走线尽量短，避免与数字信号并行走线
2. **I2C 上拉**：4.7kΩ 上拉电阻靠近 IMU 放置
3. **电源滤波**：每个 IC 的 VCC 旁放置 100nF 去耦电容
4. **电池走线**：电池正负极走线加粗，承载充放电电流
5. **BLE 天线**：XIAO 模组天线区域禁止铺铜，保持净空
6. **戒指端小型化**：戒指端 PCB 需控制在 ≤ 15mm × 15mm，考虑柔性 PCB
