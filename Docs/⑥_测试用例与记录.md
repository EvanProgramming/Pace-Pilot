## 1. 测试概述

### 1.1 测试目标

验证 BeatSync 原型的核心功能是否达到设计目标，重点验证三大核心功能：
1. 步频检测精度
2. 手势识别准确率
3. 骨传导音质可接受度

### 1.2 测试环境

| 项目 | 配置 |
|------|------|
| 测试场地 | 室内跑步机 + 户外跑道 |
| 测试人员 | 1 人（开发者本人） |
| 测试设备 | 面包板搭建原型（Phase 1）/ 完整原型（Phase 2） |
| 参考设备 | Garmin 手表（步频基准）、手机节拍器 |

### TC-005 骨传导音质主观评估

| 字段 | 内容 |
|------|------|
| 用例编号 | TC-005 |
| 测试名称 | 骨传导音质可接受度评估 |
| 前置条件 | DFPlayer + PAM8403 + 骨传导振子链路已搭建 |
| 优先级 | P0（关键决策点） |

**测试步骤**:

| 步骤 | 操作 | 评估内容 |
|------|------|---------|
| 1 | 播放 128kbps MP3 流行音乐 | 整体可听度 |
| 2 | 播放鼓点密集的电子乐 | 低频表现 |
| 3 | 播放人声为主的音乐 | 中频清晰度 |
| 4 | 播放高音丰富的音乐 | 高频表现 |
| 5 | 跑步过程中播放音乐 | 运动中稳定性、环境音感知 |
| 6 | 对比手机外放/普通入耳式耳机 | 主观对比评分 |

**主观评分标准** (1~5 分):

| 维度 | 1 分 | 3 分 | 5 分 |
|------|------|------|------|
| 清晰度 | 完全听不清 | 勉强听清歌词 | 清晰可辨 |
| 低频 | 无低频 | 有微弱低频 | 低频有力 |
| 高频 | 刺耳/缺失 | 一般 | 清亮 |
| 整体可接受度 | 无法接受 | 勉强可接受 | 令人满意 |

**测试记录**:

| 日期 | 维度 | 评分 | 备注 |
|------|------|------|------|
| | 清晰度 | /5 | |
| | 低频 | /5 | |
| | 高频 | /5 | |
| | 整体可接受度 | /5 | |

**决策标准**: 整体可接受度 ≥ 3 分 → 继续原型开发；< 3 分 → 考虑止损或升级 DAC 方案
